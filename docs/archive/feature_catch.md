# ⭐ 스타 캐치 (CatchScene) — 개발 이력 아카이브

> **분리 이유**: 레인보우 스케치북 v1.0 리디자인으로 스케치북 콘텐츠 집중 → 미니게임 별도 앱 예정  
> **완성도**: 100% 완성  
> **최종 작업일**: 2026-05

---

## 🎮 게임 개요

- **장르**: 낙하물 받기 (캐주얼 액션)
- **대상**: 5~8세
- **규칙**: 30초 동안 떨어지는 별/캔디를 바구니로 받기 → 점수 최대화
- **조작**: 터치/드래그로 바구니(토슴이) 좌우 이동
- **특징**: 별 5개 콤보 시 무지개 보너스 이벤트

## ⚙️ 주요 로직

- **낙하 아이템**: `Phaser.Physics.Arcade` 중력 적용, 랜덤 x 좌표
  - 별(⭐): 일반 아이템 (+1점)
  - 무지개 별(🌈⭐): 희귀 아이템 (+3점)
  - 폭탄(💣): 회피 아이템 (-1점)
- **바구니**: `toseumi-basket` 텍스처, 터치 입력 → x 위치 추적
- **타이머**: 30초 카운트다운 → 종료 시 결과 팝업
- **스폰 속도**: 시간 경과에 따라 점진적 가속

## 💡 별도 앱 출시 시 고려사항

- 스테이지 시스템 (30초 → 45초 → 60초 단계)
- 캐릭터 선택 (바구니 스킨)
- 아이템 다양화 (현재 5종 → 10종)
- 최고점 기록 (로컬 저장 + 랭킹)
- 파워업 아이템 (자석, 확대 바구니 등)
