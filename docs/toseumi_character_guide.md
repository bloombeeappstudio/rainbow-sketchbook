## 2. 캐릭터 외형 디테일

| 구분 | 디자인 기준 |
|---|---|
| 전체 비율 | 머리:몸 = 1:1 수준의 아기형 비율 |
| 실루엣 | 동글동글하고 폭신한 솜뭉치 형태 |
| 귀 | 짧고 둥근 토끼 귀 |
| 귀 안쪽 | 작은 별 모양 포인트 삽입 |
| 눈 | 얼굴의 약 40%를 차지하는 크고 동그란 반짝이는 눈 |
| 입 | 작고 귀여운 미소 |
| 볼 | 살짝 진한 핑크 볼터치 |
| 몸 색상 | 메인 파스텔 핑크 `#FFD1DC` |
| 배 | 흰색 또는 크림빛 흰색 |
| 발바닥 | 흰색 포인트 |
| **앞발/손 ⭐** | **둥글둥글한 단일 mitten(벙어리장갑) 형태, 손가락 분리 절대 X** |
| **뒷발 ⭐** | **둥근 토끼 발 형태, 발가락 분리 X (단일 동그란 형태)** |
| 외곽선 | 검정 라인 금지, 같은 색의 진한 톤으로 부드럽게 처리 |

> 🐰 **손/발 일관성 (필수)**: 토슴이는 토끼 캐릭터이므로 손/발이 **사람처럼 손가락 분리되면 안 됩니다**. 4개나 5개 손가락 X. 항상 둥글둥글한 단일 mitten 형태로 그려야 합니다.
>
> 영문 표기: `rounded mitten-style paws, no separated fingers, classic cute bunny paws`

## 3. 톤앤매너

- 따뜻하고 부드러운 파스텔톤
- 채도는 70% 이내
- 어둡거나 강한 대비는 지양
- 3D 렌더링이 아닌 2D 평면 일러스트 스타일
- 어린이용 모바일 게임에 적합한 안전하고 친근한 인상
- Sago Mini, 산리오 모찌풍처럼 단순하고 둥근 인상
- 너무 복잡한 장식보다, 한눈에 기억되는 귀여운 실루엣 우선

## 5. 시안별 방향

### A안. 기본 화이트 배경형

- 캐릭터 자체의 형태와 인상을 가장 명확하게 보여주는 기본 시안
- 흰색 배경 중심
- 게임 내 튜토리얼, 프로필, 아이콘, 말풍선 옆 캐릭터로 활용하기 좋음
- 별나라 설정은 귀 안쪽 별 포인트와 눈 속 반짝임으로만 은은하게 표현

### B안. 별빛 밤하늘 배경형

- 무지개 별나라에 사는 마법 토끼라는 세계관을 더 잘 보여주는 시안
- 보라빛 밤하늘, 작은 별, 파스텔 구름을 배경으로 구성
- 앱 첫 진입 화면, 온보딩, 로딩 화면, 이벤트 배너에 적합

### C안. 손 흔드는 가이드형

- 한쪽 손을 살짝 들어 아이를 맞이하는 느낌
- “안녕!”, “따라와!”, “도와줄게!” 같은 가이드 역할에 적합
- 기본 포즈보다 인터랙션성이 강함
- 튜토리얼 시작, 미션 안내, 보상 안내 화면에 활용하기 좋음

### D안. 드리미 밤하늘형

- 별빛, 달, 부드러운 후광을 활용해 더 몽글몽글하고 환상적인 느낌
- 캐릭터의 마법 토끼 콘셉트가 가장 잘 드러나는 방향
- 메인 키비주얼이나 대표 홍보 이미지에 적합

## 6. Claude 작업 요청용 프롬프트

아래 내용을 기준으로 5~8세 어린이용 모바일 게임의 가이드 캐릭터 “토슴이”를 디자인해 주세요.

```text
귀여운 분홍색 토끼 캐릭터 "토슴이"를 2D 평면 일러스트 스타일로 디자인해 주세요.

[캐릭터 컨셉]
- 이름: 토슴이
- 종족: 무지개 별나라에 사는 마법 토끼
- 역할: 5~8세 어린이용 모바일 게임 가이드 캐릭터
- 핵심 인상: 따뜻함, 귀여움, 부드러움, 친근함

[외형 디테일]
- 스타일: 동글동글한 솜뭉치 스타일, 모찌형 키즈 캐릭터
- 비율: 머리:몸 = 1:1 정도의 아기형 비율
- 귀: 짧고 둥근 형태
- 귀 안쪽: 작은 별 모양 포인트
- 눈: 얼굴의 약 40% 정도를 차지하는 크고 동그란 반짝이는 눈
- 입: 작고 귀여운 미소
- 몸 색상: 메인 파스텔 핑크 #FFD1DC
- 배와 발바닥: 흰색 또는 크림빛 흰색
- 볼터치: 살짝 진한 핑크
- 외곽선: 검정색 금지, 같은 색의 진한 톤으로 부드럽게 처리

[톤앤매너]
- 따뜻하고 부드러운 파스텔톤
- 채도는 70% 이내
- 3D가 아닌 2D 평면 일러스트
- 어린이용 모바일 게임에 적합한 안전하고 친근한 느낌
- 복잡한 장식보다 한눈에 기억되는 귀여운 실루엣 우선

[연출]

- 게임 내 튜토리얼, 온보딩, 미션 안내, 보상 안내에서 사용할 수 있는 가이드 캐릭터로 제작

[추가 요청]
- 3~4가지 시안으로 변주해 주세요.
  1. 흰색 배경의 기본 캐릭터 시안
  2. 별빛 밤하늘 배경 시안
  3. 한쪽 손을 살짝 들어 인사하는 가이드 시안
  4. 달과 별빛이 있는 몽글몽글한 대표 키비주얼 시안
```

## 7. 제작 시 주의사항

| 주의사항 | 이유 |
|---|---|
| 검정 외곽선 사용 금지 | 어린이용 앱에 맞는 부드러운 인상 유지 |
| 귀를 너무 길게 만들지 않기 | 짧고 둥근 형태가 토슴이의 차별점 |
| 눈을 너무 날카롭게 그리지 않기 | 가이드 캐릭터는 친근함이 중요 |
| 장식을 과하게 넣지 않기 | 모바일 화면에서 작게 보여도 인식 가능해야 함 |
| 3D 느낌 지양 | 현재 콘셉트는 2D 평면 일러스트 기준 |
| 색을 너무 쨍하게 쓰지 않기 | 파스텔, 위로감, 유아 친화적 톤 유지 |
