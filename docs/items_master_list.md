## 💎 3. 보석 4종 (모양)

| ID | 모양 | 키 |
|---|---|---|
| 1 | 하트 보석 | `heart` |
| 2 | 별 보석 | `star` |
| 3 | 다이아몬드 | `diamond` |
| 4 | 물방울 | `drop` |

각 보석은 4가지 색깔 변형 가능: `pink` / `purple` / `gold` / `cyan`

> **ChatGPT 가이드**: 2×2 그리드 1장으로 4종 한 번에. 각각 분홍/보라 그라데이션 + 반짝임.

## 📋 코드 동기화

이 리스트의 데이터는 다음 파일과 동일:
- 크레파스 64색: `game/src/data/colors.js`
- 스티커/보석: `game/src/systems/RewardSystem.js`

새 아이템 추가 시 **반드시** 이 문서 + 코드 동시 업데이트.
