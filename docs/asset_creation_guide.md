# 🎨 레인보우 스케치북 — 자산 제작 가이드 (v1.0)

> 이 문서를 보면서 ChatGPT에 자연어로 그대로 복붙하시면 돼요.
> 모든 자산은 **토슴이 처음 만든 ChatGPT 대화창**에서 진행 (캐릭터 일관성).

## 🎯 게임 정체성

- **게임명**: 레인보우 스케치북: 어린이 그림 색칠 게임
- **타겟**: 5~8세 어린이 + 학부모
- **세계관**: 색을 잃어버린 레인보우 별나라에 무지개를 되찾아주는 여행
- **메인 캐릭터**: 토슴이 (분홍색 토끼)
- **화면 방향**: 가로 모드 (모바일 + 태블릿)

## 🎨 톤앤매너 원칙 (모든 자산 공통)

### 시각 스타일
- **2D 평면 일러스트** (3D 렌더링 X)
- **파스텔톤** (채도 70% 이내)
- **부드러운 그라데이션**
- **검정 외곽선 사용 X** — 같은 색의 진한 톤으로 부드럽게 처리
- **동글동글한 형태** — 모든 사물이 부드럽고 안전한 곡선
- **참고 스타일**: Sago Mini, 산리오 모찌풍

### 색감 가이드
- **메인 컬러**: 파스텔 핑크 `#FFD1DC`
- **보조**: 파스텔 보라 `#B19CFF`, 노랑 `#FFD93D`, 민트 `#7FD8C0`
- **배경**: 보라/네이비 밤하늘 `#2C1B47`
- **자극적 색 X** (빨강/네온/형광 X)

### 정서 톤
- **따뜻함 / 사랑스러움 / 안전함**
- 슬픈 장면도 5~8세에게 너무 무겁지 않게
- 모든 표현은 동화책 한 페이지처럼

### 손/발 표현 (사용자님 강조 포인트)

>
> ```

> 동글동글 폭신한 토끼 인형/봉제인형처럼 표현해줘.

> 통통하고 부드러운 솜뭉치 같은 느낌.
> 이전에 만든 토슴이 손과 동일한 스타일.
> ```

## 📐 해상도 정책

ChatGPT(DALL-E 3)가 지원하는 3가지 해상도:

| 비율 | 해상도 | 우리 게임에서 |
|---|---|---|
| 가로 16:9 | **1792 × 1024** | 풀 화면 배경/씬 (그림책 컷, 게임 배경) |
| 정사각 1:1 | **1024 × 1024** | 단일 캐릭터/오브젝트, 그리드 묶음 |
| 세로 9:16 | 1024 × 1792 | 사용 X (가로 모드 게임) |

**프롬프트에 명시할 것**:
- 가로 자산: `"가로 16:9 wide format"` 또는 `"landscape orientation"`
- 정사각 자산: `"square 1:1 format"` 또는 `"흰 배경의 정사각 일러스트"`

태블릿 대응: ChatGPT 1792×1024 해상도면 충분 (Galaxy Tab/iPad 모두 자동 스케일).

### 9. 바구니 든 토슴이 (스타 캐치용)
- **파일명**: `toseumi-basket.png`
- **프롬프트**:
```

위를 올려다보는 모습 (1024x1024 정사각).

외형 일관성 유지, 손/발은 동글동글 폭신한 paw 모양.

표현:
- 토슴이가 정면, 두 앞발(paw)로 바구니를 가슴 앞에 들고 있음
- 바구니: 부드러운 분홍 또는 파스텔 노란색, 손잡이가 있는 등나무 바구니 느낌,
  바구니도 동글동글한 토끼 인형 톤에 맞게 부드러운 형태

### 일관성 유지
1. **같은 ChatGPT 대화창**에서 전부 진행 (가장 중요!)
