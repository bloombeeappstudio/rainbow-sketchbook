## 1. 개요

레인보우 스케치북(이하 "본 앱")은 5세 이상 어린이를 위한 색칠/그림 그리기 모바일 게임입니다. 본 앱은 어린이 사용자의 안전과 사생활을 최우선으로 합니다.

본 방침은 한국 「개인정보 보호법」, 미국 「아동 온라인 사생활 보호법(COPPA)」, 유럽연합 「일반 데이터 보호 규정(GDPR)」 제8조(어린이 개인정보 처리 특례) 및 Google Play 「Families 정책」을 준수합니다.

## 3. 데이터 저장 위치

본 앱의 모든 데이터(색칠한 작품, 진행 상황, 잠금 해제 상태 등)는 **사용자 기기 내부 저장소**에만 저장됩니다. 외부 서버로 전송되지 않습니다.

- 저장 위치: Android 앱 전용 저장소 (`/data/data/com.rainbow.sketchbook/`)
- 저장 방식: 로컬 키-값 저장소 (Capacitor Preferences / localStorage)
- 외부 전송: **없음**

## 4. 인앱 결제 처리

본 앱은 1회성 인앱 결제 "별빛 아티스트 팩"(₩2,500)을 제공합니다.

- 결제는 **Google Play 결제 시스템**을 통해서만 처리됩니다.
- 결제 과정에서 입력되는 결제 정보(카드 번호, 청구지 등)는 Google이 직접 처리하며, **본 앱은 결제 정보에 접근하지 않습니다**.
- 결제 영수증은 Google Play로부터 받아 잠금 해제 처리에만 사용되며, 외부로 전송되지 않습니다.

자세한 사항은 Google 개인정보처리방침을 참조해 주세요: <https://policies.google.com/privacy>

## 7. 외부 SDK / 라이브러리

본 앱이 사용하는 외부 라이브러리 및 정보 수집 여부:

| 라이브러리 | 용도 | 데이터 수집 |
|---|---|---|

### Firebase Analytics 어린이 모드

서비스 품질 향상을 위해 다음 **익명 데이터**만 수집합니다:
- 앱 실행 횟수, 세션 시간, 화면 전환
- 도안 선택/색칠 완성/그림 완성 등 사용 행동 (개인 식별 X)
- 결제 발생 (금액 통계용, 카드 정보 X)
- 갤러리 저장/공유 사용 빈도

다음은 **수집하지 않습니다**:
- 광고 식별자(AAID): AndroidManifest에서 명시적으로 거부
- 개인화 광고 타겟팅: Firebase Console에서 비활성화
- 사용자 식별 정보: 익명 ID(Firebase Installations ID)만 사용
- 위치 정보: 미수집

**광고는 일절 표시하지 않습니다.** Firebase 데이터는 오직 우리 앱의 품질 개선에만 사용되며, 제3자 광고 네트워크와 공유되지 않습니다.

데이터 보관 기간: **14개월** (Firebase Console 최대 설정 — 장기 사용 통계 활용)

## 7-1. 사용자 작품 공유 (선택)

이용자는 본인이 만든 작품을 다음 두 방식으로 활용할 수 있습니다:
- **핸드폰 사진 앨범 저장**: 본 앱이 자체 앨범("레인보우 스케치북")을 만들어 PNG 저장 (외부 서버 전송 X)
- **다른 앱으로 공유**: 카카오톡, 메일 등 시스템 공유 시트를 통해 외부 앱으로 전송 (사용자 명시적 선택 + 부모 확인 필요)

공유 기능 사용 시:
- 작품 PNG 파일만 전송됨 (개인정보 X)
- 외부 공유 전 **부모님 확인 절차**(수학 문제)를 거쳐 어린이의 우발적 공유를 방지
