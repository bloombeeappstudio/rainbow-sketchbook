4. 이미지에 텍스트/글자/숫자 절대 그리지 마. 그림만.
5. 한 번에 한 자산만 (여러 컷을 한 장에 모으지 마)

## 💡 사용 팁

1. **순서 권장**: 1번부터 순서대로 (그림책 5컷 먼저 끝내고 → 캐릭터 → 자산)
2. **같은 ChatGPT 대화창**에서 (토슴이 일관성)
3. **명령 컬럼만 복사** → ChatGPT에 그대로 붙여넣기
4. **결과 마음에 안 들면**: "다시" / "더 사랑스럽게" / "손가락 없이 mitten 모양으로"
5. **그리드 자산** (6, 9, 10, 13, 15, 16, 17번)은 한 장으로 받으시고 → 알려주시면 제가 분할
