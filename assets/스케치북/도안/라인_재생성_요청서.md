## 📋 ChatGPT 재생성 프롬프트 (복사해서 사용)

```
어린이용(5-8세) 색칠 도안을 다시 만들어 주세요.

## ⚠️ 가장 중요한 요구사항 (라인 마감)
- **모든 외곽 라인은 완전히 닫혀(closed) 있어야 합니다.** 끊긴 곳/뚫린 곳 절대 X
- 라인 두께는 균일하게 4-6px (가는 곳 1px X)
- 라인 색은 순수 검정 (#000000), anti-alias 최소화
- 각 영역(머리/몸통/팔/다리/꼬리 등)의 경계가 명확하게 분리되어 있어야 함
- 디지털로 확대해서 봤을 때 라인이 1픽셀이라도 끊겨 보이면 안 됩니다
- 영역과 영역이 만나는 지점(예: 팔과 몸통 사이)에서 라인이 V자 또는 ∨자로 완전히 합쳐져야 함

## 도안 구성
- 흰 배경 (#FFFFFF), 검정 외곽선만
- 캐릭터/오브젝트는 캔버스 중앙에 70-85% 크기로 배치
- 좌측 상단에 작은 컬러 샘플 (참고용, 5-10% 크기)
- 우측 상단 또는 빈 공간에 보조 오브젝트 1-2개 (배경 풍경)
- 모든 보조 오브젝트도 라인 완전히 닫혀 있어야 함

## 금지 사항
- ❌ 라인이 1픽셀이라도 끊기는 것
- ❌ 영역 안에 패턴/점박이/줄무늬 (색칠하기 어려움)
- ❌ 너무 가는 라인 (1-2px)
- ❌ 라인 색이 회색 (#333, #555 등) — 반드시 순수 검정
- ❌ 그라데이션 배경

## 출력 (게임 인게임 스펙 — 반드시 16:9 가로)
- **1순위: `1792 × 1024` PNG (16:9 가로 landscape, DALL-E 3 16:9 옵션)**
- 차선: `1536 × 1024` PNG (3:2, GPT-Image-1 기본) — 코드가 자동 16:9 변환
- ❌ **1:1 정사각형 (1024×1024) 절대 금지** — 게임 16:9 캔버스에서 좌우 여백 낭비
- 투명 배경 X (흰 배경 #FFFFFF 유지)
- 라인만 순수 검정 (#000000), 모든 영역 안쪽은 흰색
- 게임 내부 표준: `1672 × 941` (후처리에서 자동 리사이즈됨, 대표님은 ChatGPT 받은 원본만 폴더에 넣으면 됨)

## 검증 요청
생성 후 도안을 200% 확대해서 다음을 확인해주세요:
1. 모든 영역의 라인이 끊김 없이 완전히 닫혀있는가?
2. 라인 두께가 4-6px로 균일한가?
3. 영역과 영역이 만나는 모서리에서 라인이 정확히 합쳐지는가?

위 3가지가 모두 OK일 때만 결과를 보여주세요.
```

## 🔧 코드 측 안전망 (현재 상태)

라인이 완전히 깔끔하지 않아도 안전망 3중 작동:
1. binarize 임계값 500 — anti-alias 가장자리 회색까지 검정 처리
2. region mask flood-fill 흰 픽셀만 통과 (R/G/B ≥235)
3. mask 1픽셀 erosion — 라인 가장자리 안전 마진

이 안전망으로 **작은 anti-alias 누수는 자동 차단**되지만, 도안 라인이 명백히 끊긴 경우(1픽셀 이상)는 도안 PNG 단계에서 잡아야 함.
